# تصاویر راهنمای نقشه

این پوشه برای اسکرین‌شات‌های راهنمای نقشه در صفحه اصلی است.

مسیر در پروژه:

```
nextjs/public/images/map-guide/
```

URL در مرورگر: `/images/map-guide/نام-فایل`

## فایل‌های پیشنهادی

| فایل | موضوع |
|------|--------|
| `01-overview.png` | نمای کلی نقشه با مارکرها |
| `02-fullscreen.png` | دکمه بزرگنمایی و حالت تمام‌صفحه |
| `03-layer-toggle.png` | سوییچ «متخصصین / کارها» |
| `04-filters.png` | فیلتر دسته، زیردسته، بخش، محله |
| `05-markers.png` | مارکر و پاپ‌آپ بعد از کلیک |
| `06-expert-scope.png` | «کارهای تخصص من / همه کارها» |
| `07-map-settings.png` | نوار 2D / 3D / مرزها |
| `08-corner-controls.png` | دکمه‌های لایه و i گوشه نقشه |
| `09-mobile-sheet.png` | شیت فیلتر پایین در موبایل |

فرمت PNG یا JPG (ترجیحاً عرض 1200px). تا وقتی فایل را نگذارید، جای خالی با نام فایل در راهنما نشان داده می‌شود.
